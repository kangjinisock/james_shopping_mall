# Shopping Mall Server

Node.js + Express + MongoDB 조합의 시작 템플릿입니다.

## 설치

```bash
npm install
```

## 실행

```bash
npm run dev
```

또는

```bash
npm start
```

## 환경 변수 (선택)

- PORT (기본값: 4000)
- MONGODB_ATLAS_URL (우선 사용, 값이 없으면 로컬 주소 사용)
- DB_NAME (기본값: shopping-mall)

## 기본 엔드포인트

- GET /
- GET /health
